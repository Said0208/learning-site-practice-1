# 🌐 Web.Dev Landing Page

A modern responsive landing page for a web development agency.
This project showcases services, portfolio, testimonials, pricing plans, and contact information.

---

## 🚀 Features

* Responsive Header Navigation
* Hero Banner Section
* Company Partners Section
* Specialization / Services Section
* Portfolio Gallery
* Customer Testimonials
* Pricing Plans
* Contact Call-to-Action
* Footer with social links and newsletter input
* Animated buttons and cards

---

## 🛠 Technologies Used

* HTML5
* CSS3
* BEM Methodology (Block Element Modifier)
* SVG Icons
* Responsive Layout

3. Open `index.html` in your browser.
