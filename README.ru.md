# 🌐 Web.Dev — Лендинг веб-студии

Современный адаптивный лендинг для агентства веб-разработки.
Проект демонстрирует услуги, портфолио, отзывы клиентов, тарифные планы и контактную информацию.

---

## 🚀 Возможности

* Адаптивная шапка с навигацией
* Главный баннер (Hero Section)
* Блок с компаниями-партнёрами
* Раздел «Наши услуги»
* Галерея портфолио
* Отзывы клиентов
* Тарифные планы
* Блок «Связаться с нами»
* Подвал с социальными сетями и подпиской
* Анимированные кнопки и карточки

---

## 🛠 Используемые технологии

* HTML5
* CSS3
* Методология BEM (Block Element Modifier)
* SVG-иконки
* Адаптивная вёрстка

## 📸 Обзор разделов

### 🏠 Шапка (Header)

Навигационное меню:

* Главная
* О нас
* Отзывы
* Контакты
* Кнопки «Войти» и «Регистрация»

### 🎯 Баннер

Главный экран с:

* Заголовком
* Описанием
* Кнопками действия
* Декоративными изображениями

### 💼 Наши специализации

Предоставляемые услуги:

* Разработка на Webflow
* Разработка на Shopify
* Веб-дизайн в Figma
* Разработка на PHP

### 🖼 Портфолио

Примеры проектов:

* Модный лендинг
* Здоровье и фитнес
* Магазин мебели
* IT-решения

### ⭐ Отзывы

Отзывы клиентов с рейтингом.

### 💰 Тарифы

Три тарифных плана:

* Starter
* Teams
* Company (Индивидуальный)

### 📞 Контактный блок

Призыв к действию для связи с компанией.

### 🔻 Подвал (Footer)

* Ссылки на социальные сети
* Полезные ресурсы
* Информация о компании
* Форма подписки на email

---

## ▶️ Как запустить

1. Скачайте или клонируйте проект.
2. Убедитесь, что все изображения находятся в папке `img`.
3. Откройте файл `index.html` в браузере.
